Chapter 2-6

그래프 기초

matplotlib 기본 구조와 선 그래프, 막대그래프, 산점도, 히스토그램의 용도를 학습합니다.

Chapter 2이전다음

1. 그래프가 필요한 이유

숫자 표만 보면 데이터의 패턴을 빠르게 파악하기 어렵습니다. 그래프는 변화, 비교, 관계, 분포를 시각적으로 보여주어 분석 결과를 더 쉽게 설명하게 해줍니다.

선 그래프시간에 따른 변화를 확인합니다.
막대그래프그룹별 크기를 비교합니다.
산점도두 변수의 관계를 확인합니다.
히스토그램값의 분포를 확인합니다.

2. matplotlib 기본 구조

import matplotlib.pyplot as plt x = [1, 2, 3, 4] y = [10, 20, 15, 30] plt.plot(x, y) plt.show()

plt.show()는 작성한 그래프를 화면에 표시합니다.

3. 선 그래프

선 그래프는 날짜별 매출, 월별 방문자 수처럼 시간 흐름에 따른 변화를 볼 때 적합합니다.

days = ["월", "화", "수", "목", "금"] sales = [120000, 150000, 130000, 180000, 210000] plt.plot(days, sales, marker="o") plt.title("요일별 매출") plt.xlabel("요일") plt.ylabel("매출") plt.show()

4. 막대그래프

막대그래프는 카테고리별 매출, 지점별 주문 수처럼 항목 간 크기를 비교할 때 사용합니다.

menus = ["커피", "에이드", "푸드"] sales = [350000, 120000, 180000] plt.bar(menus, sales) plt.title("카테고리별 매출") plt.show()

5. 산점도

산점도는 광고비와 매출, 공부 시간과 점수처럼 두 숫자형 변수 사이의 관계를 볼 때 사용합니다.

ad_cost = [10, 20, 30, 40, 50] sales = [100, 180, 260, 300, 380] plt.scatter(ad_cost, sales) plt.xlabel("광고비") plt.ylabel("매출") plt.show()

6. 히스토그램

히스토그램은 점수, 구매금액, 이용시간처럼 값이 어느 구간에 많이 몰려 있는지 확인합니다.

scores = [60, 70, 75, 80, 82, 85, 90, 95, 100] plt.hist(scores, bins=5) plt.title("점수 분포") plt.show()

7. 실습 체크리스트

  • 그래프 종류별 용도를 설명할 수 있다.
  • 선 그래프를 그릴 수 있다.
  • 막대그래프를 그릴 수 있다.
  • 산점도를 그릴 수 있다.
  • 히스토그램을 그릴 수 있다.

8. 종합 실습 문제

1. 문제 출제

요일별 매출 데이터를 사용해 선 그래프와 막대그래프를 각각 작성하세요.

  • 요일과 매출 리스트를 만듭니다.
  • 선 그래프로 매출 변화를 표현합니다.
  • 막대그래프로 요일별 매출 크기를 비교합니다.
  • 각 그래프에 제목과 축 이름을 추가합니다.

2. 개념 요약

개념활용
plot변화 추이 표시
bar항목별 비교
title/xlabel/ylabel그래프 설명 추가

3. 수도코드

요일 리스트를 만든다. 매출 리스트를 만든다. 선 그래프를 그린다. 제목과 축 이름을 추가한다. 그래프를 출력한다. 막대그래프를 그린다. 제목과 축 이름을 추가한다. 그래프를 출력한다.

4. 실행 예시

입력 데이터 예시는 다음과 같습니다.

days = ["월", "화", "수", "목", "금"] sales = [120000, 150000, 130000, 180000, 210000]
