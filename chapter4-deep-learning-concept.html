1. 딥러닝이란?

딥러닝은 여러 층의 신경망을 사용해 데이터의 복잡한 패턴을 학습하는 머신러닝 방법입니다. 이미지, 음성, 자연어처럼 규칙을 사람이 직접 만들기 어려운 데이터에서 특히 많이 사용됩니다.

뉴런입력값을 받아 계산한 뒤 출력값을 만듭니다.
가중치입력값이 결과에 미치는 영향의 크기입니다.
층뉴런이 모인 계산 단계입니다.
학습오차를 줄이도록 가중치를 조정합니다.

2. 뉴런의 직관

뉴런은 여러 입력값을 받아 중요도에 따라 곱하고 더한 뒤, 활성화 함수를 통과시켜 결과를 만듭니다.

입력값 × 가중치 → 합계 계산 → 활성화 함수 → 출력값
쉽게 말해: 뉴런은 여러 단서를 종합해 하나의 판단 점수를 만드는 작은 계산기입니다.

5. 실습 체크리스트

  • 딥러닝과 일반 머신러닝의 차이를 설명할 수 있다.
  • 뉴런, 가중치, 층의 의미를 말할 수 있다.
  • 입력층, 은닉층, 출력층의 역할을 구분할 수 있다.
  • 케라스 모델 생성 흐름을 순서대로 설명할 수 있다.

6. 종합 실습 문제

1. 문제 출제

고객 데이터 3개 입력값을 받아 구매 여부를 예측하는 신경망 구조를 설계해 보세요. 실제 학습보다 구조 설명에 초점을 둡니다.

  • 입력값은 방문횟수, 장바구니 금액, 이전 주문 수입니다.
  • 출력값은 구매 확률 1개입니다.
  • 은닉층은 1개 이상 사용합니다.
  • 각 층의 역할을 문장으로 설명합니다.

2. 개념 요약

시그모이드0~1 사이 확률로 변환합니다.

3. 수도코드

입력 특성 개수를 확인한다. Sequential 모델을 만든다. 은닉층을 추가한다. 출력층을 추가한다. 분류 문제에 맞는 손실 함수와 평가 지표를 정한다. 모델 구조를 문장으로 해석한다.
