# 9. 종합 실습 문제: 도서관리 업무 데이터 분석

## 1. 문제 출제

도서관의 도서 대출 데이터를 이용해 **우선 관리가 필요한 도서 목록**을 추출하여 새 CSV 파일로 저장하는 프로그램을 작성해 보세요.

도서관에서는 다음과 같은 기준으로 도서를 관리하려고 합니다.

- `books.csv` 파일을 읽습니다.
- 대출 횟수 `borrow_count`가 **30회 이상**인 도서만 선택합니다.
- 남은 재고 수량 `stock`이 **3권 이하**인 도서만 선택합니다.
- `title`, `author`, `borrow_count`, `stock` 컬럼만 남깁니다.
- 결과를 `priority_books.csv` 파일로 저장합니다.

이 문제는 약 **1시간 정도** 소요되는 종합 실습 과제입니다.

---

## 2. 문제 해결에 필요한 개념 요약

| 개념 | 활용 방법 |
|---|---|
| CSV 읽기 | `pd.read_csv()`로 도서 데이터를 읽습니다. |
| DataFrame | CSV 데이터를 표 형태로 다룹니다. |
| 행 필터링 | 대출 횟수와 재고 수량 조건을 동시에 적용합니다. |
| 비교 연산자 | `>=`, `<=`를 사용해 조건을 만듭니다. |
| 논리 연산자 | `&`를 사용해 두 조건을 모두 만족하는 데이터를 찾습니다. |
| 열 선택 | 필요한 컬럼만 선택합니다. |
| CSV 저장 | `to_csv()`로 결과 파일을 저장합니다. |

## 3. 수도코드

pandas를 불러온다.

`books.csv` 파일을 읽는다.

`borrow_count`가 30 이상인지 조건을 만든다.

`stock`이 3 이하인지 조건을 만든다.

두 조건을 모두 만족하는 행만 선택한다.

`title`, `author`, `borrow_count`, `stock` 컬럼만 선택한다.

결과를 화면에 출력한다.

`priority_books.csv` 파일로 저장한다.

## 6. 정답 코드 작성 조건

아래 조건을 모두 만족하도록 코드를 작성하세요.

1. `pandas`를 사용해야 합니다.
2. `books.csv` 파일을 읽어야 합니다.
3. `borrow_count >= 30` 조건을 사용해야 합니다.
4. `stock <= 3` 조건을 사용해야 합니다.
5. 두 조건을 모두 만족하는 도서만 선택해야 합니다.
6. 결과에는 다음 컬럼만 포함해야 합니다.

```python
["title", "author", "borrow_count", "stock"]
```

7. 결과를 `priority_books.csv` 파일로 저장해야 합니다.
8. CSV 저장 시 인덱스 번호는 저장하지 않습니다.

---

## 7. 힌트

두 조건을 동시에 사용할 때는 아래와 같은 구조를 사용합니다.

```python
조건1 = df["컬럼명"] >= 기준값
조건2 = df["컬럼명"] <= 기준값

result = df[조건1 & 조건2]
```

필요한 컬럼만 선택할 때는 아래와 같은 구조를 사용합니다.

```python
result = result[["컬럼1", "컬럼2", "컬럼3"]]
```

CSV 파일로 저장할 때는 아래와 같은 구조를 사용합니다.

```python
result.to_csv("파일명.csv", index=False, encoding="utf-8-sig")
```
