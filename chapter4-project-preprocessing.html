1. 이번 단계의 목표

프로젝트 기획에서 정한 데이터셋을 실제로 불러오고, 분석 가능한 형태로 정리합니다. 이후 그래프로 데이터의 특징을 확인합니다.

데이터 로딩CSV 또는 엑셀 데이터를 읽습니다.
데이터 점검행/열, 자료형, 결측치를 확인합니다.
전처리결측치, 이상치, 필요한 컬럼을 정리합니다.
시각화패턴을 그래프로 확인합니다.

2. 기본 점검 코드

import pandas as pd import matplotlib.pyplot as plt df = pd.read_csv("project_data.csv") print(df.head()) print(df.shape) print(df.info()) print(df.isna().sum())

처음 데이터를 읽은 뒤에는 바로 모델을 만들지 말고, 데이터 구조와 품질을 먼저 확인해야 합니다.

3. 결측치와 이상치 처리

결측치는 비어 있는 값이고, 이상치는 일반적인 범위에서 벗어난 값입니다. 입문 단계에서는 결측치 개수와 기초 통계를 확인하는 것부터 시작합니다.

print(df.describe()) df["cart_amount"] = df["cart_amount"].fillna(0) df = df[df["cart_amount"] >= 0]
주의: 이상치를 무조건 삭제하지 말고, 실제 업무상 가능한 값인지 먼저 확인해야 합니다.

4. 그룹화와 요약

카테고리나 고객 등급별로 데이터를 요약하면 전체 데이터의 특징을 빠르게 파악할 수 있습니다.

summary = df.groupby("membership")["cart_amount"].agg(["count", "mean", "sum"]) print(summary)

5. 시각화

요약 결과는 그래프로 표현하면 더 쉽게 설명할 수 있습니다.

summary["sum"].plot(kind="bar") plt.title("회원 등급별 장바구니 총액") plt.xlabel("회원 등급") plt.ylabel("총액") plt.show()

6. 실습 체크리스트

  • 프로젝트 데이터를 불러올 수 있다.
  • 데이터 구조와 결측치를 확인할 수 있다.
  • 기초 통계로 이상치를 의심할 수 있다.
  • 그룹별 요약표를 만들 수 있다.
  • 요약 결과를 그래프로 표현할 수 있다.

7. 종합 실습 문제

1. 문제 출제

고객 데이터에서 회원 등급별 장바구니 금액을 분석하고 그래프로 표현하세요.

  • project_data.csv 파일을 읽습니다.
  • 결측치 개수를 확인합니다.
  • cart_amount 결측치는 0으로 대체합니다.
  • 회원 등급별 평균과 총합을 계산합니다.
  • 회원 등급별 총합을 막대그래프로 표현합니다.

2. 문제 해결에 필요한 개념 요약

개념활용 방법
read_csv프로젝트 데이터 읽기
isna결측치 확인
fillna결측치 대체
groupby회원 등급별 집계
plot그래프 작성

3. 수도코드

CSV 파일을 읽는다. 데이터 구조를 확인한다. 결측치 개수를 확인한다. cart_amount 결측치를 0으로 대체한다. membership 기준으로 cart_amount 평균과 총합을 계산한다. 집계 결과를 출력한다. 회원 등급별 총합 막대그래프를 그린다. 그래프에서 보이는 특징을 문장으로 작성한다.

4. 실행 예시

membership count mean sum basic 20 18500 370000 silver 15 32000 480000 gold 10 58000 580000 해석: gold 등급은 고객 수는 적지만 장바구니 총액이 가장 높습니다.
