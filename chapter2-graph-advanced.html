Chapter 2-7

그래프 응용

그래프 꾸미기, 여러 그래프 비교, 분석 결과 시각화 표현 방법을 학습합니다.

Chapter 2이전다음

1. 보기 좋은 그래프가 필요한 이유

그래프는 단순히 그리는 것보다 해석하기 쉽게 만드는 것이 중요합니다. 제목, 축 이름, 범례, 값 표시가 적절히 들어가면 다른 사람이 분석 결과를 더 쉽게 이해할 수 있습니다.

제목그래프가 말하는 핵심 주제를 보여줍니다.
축 이름x축과 y축의 의미를 설명합니다.
범례여러 선이나 막대의 의미를 구분합니다.
값 표시중요 수치를 직접 보여줍니다.

2. 그래프 기본 꾸미기

import matplotlib.pyplot as plt months = ["1월", "2월", "3월"] sales = [300, 450, 380] plt.plot(months, sales, marker="o", label="매출") plt.title("월별 매출 변화") plt.xlabel("월") plt.ylabel("매출") plt.legend() plt.grid(True) plt.show()

3. 여러 데이터 비교하기

두 개 이상의 값을 비교할 때는 여러 선을 한 그래프에 그릴 수 있습니다.

months = ["1월", "2월", "3월"] sales_a = [300, 450, 380] sales_b = [280, 420, 500] plt.plot(months, sales_a, marker="o", label="A지점") plt.plot(months, sales_b, marker="o", label="B지점") plt.title("지점별 월별 매출 비교") plt.legend() plt.show()

4. 막대그래프에 값 표시하기

막대 위에 값을 직접 표시하면 보고서에서 결과를 더 명확하게 전달할 수 있습니다.

branches = ["강남점", "분당점", "수원점"] sales = [350, 280, 310] plt.bar(branches, sales) plt.title("지점별 매출") for i, value in enumerate(sales): plt.text(i, value, str(value), ha="center", va="bottom") plt.show()

5. 분석 메시지와 연결하기

그래프를 그린 뒤에는 “그래프가 무엇을 말하는지”를 문장으로 정리해야 합니다. 시각화는 결과물이 아니라 해석을 돕는 도구입니다.

그래프 관찰해석 문장 예시
3월 매출이 가장 높다3월 프로모션 이후 매출이 증가한 것으로 보입니다.
A지점이 꾸준히 높다A지점은 안정적인 매출 기반을 가진 것으로 판단됩니다.
특정 값이 매우 낮다해당 기간의 운영 이슈나 데이터 누락 가능성을 점검해야 합니다.

6. 실습 체크리스트

  • 그래프에 제목과 축 이름을 추가할 수 있다.
  • 범례를 사용해 여러 데이터를 구분할 수 있다.
  • 막대그래프에 값을 표시할 수 있다.
  • 그래프 결과를 해석 문장으로 정리할 수 있다.

7. 종합 실습 문제

1. 문제 출제

두 지점의 월별 매출을 하나의 선 그래프로 비교하고, 지점별 총매출을 막대그래프로 표현하세요.

  • A지점과 B지점의 월별 매출을 선 그래프로 비교합니다.
  • 범례, 제목, 축 이름을 추가합니다.
  • 각 지점의 총매출을 계산합니다.
  • 총매출을 막대그래프로 표현하고 막대 위에 값을 표시합니다.
  • 그래프에서 알 수 있는 점을 2문장으로 작성합니다.

2. 개념 요약

개념활용
plot월별 변화 비교
legend지점 구분
sum총매출 계산
bar/text막대그래프와 값 표시

3. 수도코드

월 리스트를 만든다. A지점 매출 리스트를 만든다. B지점 매출 리스트를 만든다. 두 지점의 선 그래프를 그린다. 제목, 축 이름, 범례를 추가한다. A지점과 B지점 총매출을 계산한다. 총매출 막대그래프를 그린다. 막대 위에 값을 표시한다. 해석 문장 2개를 작성한다.

4. 실행 예시

A지점 총매출: 1130 B지점 총매출: 1200 해석 1: B지점의 총매출이 A지점보다 높습니다. 해석 2: A지점은 2월에 가장 높은 매출을 보였고, B지점은 3월에 크게 증가했습니다.
