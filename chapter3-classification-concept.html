Chapter 3-6

지도학습: 분류 개념

2클래스 분류, 결정 경계, 확률 기반 예측, 로지스틱 회귀의 기본 개념을 이해합니다.

Chapter 3이전다음

1. 분류란?

분류는 결과가 정해진 범주 중 하나로 나오는 문제입니다. 예를 들어 고객이 구매할지, 메일이 스팸인지, 학생이 합격인지 불합격인지 판단하는 문제입니다.

입력값 X판단에 사용하는 정보
정답 y클래스 또는 라벨
확률각 클래스일 가능성
예측 클래스가장 가능성이 높은 결과

2. 2클래스 분류

2클래스 분류는 결과가 두 가지 중 하나인 문제입니다.

문제클래스 0클래스 1
구매 여부미구매구매
합격 여부불합격합격
이탈 여부유지이탈

3. 결정 경계

결정 경계는 모델이 클래스를 나누는 기준선입니다. 예를 들어 점수와 출석률을 보고 합격/불합격을 나눈다면, 어느 기준선을 중심으로 결과가 갈릴 수 있습니다.

직관: 분류 모델은 데이터를 보고 “어느 쪽에 더 가까운가?”를 판단합니다.

4. 확률 기반 예측

분류 모델은 보통 클래스에 속할 확률을 계산한 뒤, 기준값보다 높으면 특정 클래스로 판단합니다.

구매 확률 = 0.82 구매 확률 >= 0.5 이면 → 구매 구매 확률 < 0.5 이면 → 미구매

5. 실습 체크리스트

  • 분류 문제가 무엇인지 설명할 수 있다.
  • 2클래스 분류 예시를 들 수 있다.
  • 결정 경계의 의미를 설명할 수 있다.
  • 확률을 기준으로 클래스를 판단하는 방식을 이해할 수 있다.

6. 종합 실습 문제

1. 문제 출제

아래 사례에서 입력값, 정답 라벨, 클래스 의미를 정리해 보세요.

  • 출석률과 과제점수로 수료 여부 예측
  • 방문횟수와 장바구니 금액으로 구매 여부 예측
  • 온도와 습도로 불량 여부 예측

2. 개념 요약

3. 수도코드

사례를 읽는다. 예측 결과가 무엇인지 찾는다. 예측 결과가 가질 수 있는 클래스를 정리한다. 입력값으로 사용할 수 있는 정보를 정리한다. 분류 문제인 이유를 한 문장으로 작성한다.
