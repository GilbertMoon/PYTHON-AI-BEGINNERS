Chapter 3-4

회귀 모델 실습

선형 회귀 모델을 학습시키고, 예측 결과와 실제값을 비교하며 오차를 해석합니다.

Chapter 3이전다음

1. 실습 목표

간단한 매출 데이터를 사용해 회귀 모델을 학습하고, 광고비를 입력하면 예상 매출을 예측하는 흐름을 실습합니다.

데이터 준비입력 X와 목표 y를 나눕니다.
모델 생성선형 회귀 모델을 만듭니다.
모델 학습fit()으로 규칙을 찾습니다.
예측과 평가predict()와 오차를 확인합니다.

2. 데이터 준비

광고비와 매출 데이터가 있다고 가정합니다. 광고비는 입력값 X, 매출은 목표값 y입니다.

import pandas as pd df = pd.DataFrame({ "ad_cost": [10, 20, 30, 40, 50], "sales": [100, 180, 260, 320, 410] }) X = df[["ad_cost"]] y = df["sales"]
주의: scikit-learn에서 입력값 X는 보통 2차원 형태로 전달합니다. 그래서 df[["ad_cost"]]처럼 대괄호를 두 번 사용합니다.

3. 모델 학습

from sklearn.linear_model import LinearRegression model = LinearRegression() model.fit(X, y)

fit()은 입력값과 목표값의 관계를 모델이 학습하는 단계입니다.

4. 예측하기

pred = model.predict(X) print(pred)

학습에 사용한 광고비를 다시 넣어 예측값을 확인할 수 있습니다. 새로운 광고비에 대한 예측도 가능합니다.

new_data = pd.DataFrame({"ad_cost": [60]}) new_pred = model.predict(new_data) print(new_pred)

5. 오차 확인

from sklearn.metrics import mean_squared_error mse = mean_squared_error(y, pred) print("MSE:", mse)

평균제곱오차는 실제 매출과 예측 매출의 차이를 요약한 값입니다.

6. 실습 체크리스트

  • 입력값 X와 목표값 y를 나눌 수 있다.
  • 선형 회귀 모델을 생성할 수 있다.
  • fit()으로 모델을 학습할 수 있다.
  • predict()로 예측할 수 있다.
  • 평균제곱오차를 계산할 수 있다.

7. 종합 실습 문제

1. 문제 출제

광고비와 방문자 수를 입력값으로 사용하여 매출을 예측하는 회귀 모델을 만들어 보세요.

  • 입력 컬럼은 ad_cost, visitors입니다.
  • 목표 컬럼은 sales입니다.
  • 모델을 학습한 뒤 예측값을 출력합니다.
  • MSE를 계산합니다.

2. 개념 요약

개념활용
입력 X광고비와 방문자 수
목표 y매출
LinearRegression회귀 모델 생성
MSE예측 오차 확인

3. 수도코드

데이터프레임을 만든다. 입력 X와 목표 y를 분리한다. 선형 회귀 모델을 만든다. 모델을 학습한다. 학습 데이터에 대해 예측한다. 실제값과 예측값을 비교한다. MSE를 계산한다.

4. 실행 예시

예측값: [105.2 182.4 255.8 329.1 407.5] MSE: 42.36
