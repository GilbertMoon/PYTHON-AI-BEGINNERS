Chapter 3-8

회귀·분류 종합 실습

하나의 데이터셋으로 문제를 정의하고, 회귀와 분류 모델 중 적절한 모델을 선택하여 학습·평가·해석합니다.

Chapter 3이전세미프로젝트

1. 실습 목표

같은 고객 데이터에서도 목표값을 무엇으로 잡느냐에 따라 회귀 문제가 될 수도 있고 분류 문제가 될 수도 있습니다. 이번 실습은 문제 정의가 모델 선택보다 먼저라는 점을 이해하는 데 목적이 있습니다.

문제 정의예측하고 싶은 목표 결정
모델 선택회귀 또는 분류 선택
평가 지표 선택MSE 또는 정확도 선택
결과 해석업무 문장으로 정리

2. 데이터 예시

customer_id,visit_count,cart_amount,previous_orders,next_month_sales,purchased 1,2,12000,1,25000,0 2,5,45000,3,70000,1 3,1,8000,0,10000,0 4,7,60000,4,90000,1 5,3,25000,2,40000,1

3. 회귀 문제로 정의하기

next_month_sales를 예측하면 회귀 문제입니다.

X = df[["visit_count", "cart_amount", "previous_orders"]] y = df["next_month_sales"]

평가 지표로는 MSE, RMSE, MAE 등을 사용할 수 있습니다.

4. 분류 문제로 정의하기

purchased를 예측하면 분류 문제입니다.

X = df[["visit_count", "cart_amount", "previous_orders"]] y = df["purchased"]

평가 지표로는 정확도, 혼동행렬 등을 사용할 수 있습니다.

5. 실습 체크리스트

  • 같은 데이터에서 목표값에 따라 문제 유형이 달라짐을 설명할 수 있다.
  • 회귀 문제의 목표값과 평가 지표를 선택할 수 있다.
  • 분류 문제의 목표값과 평가 지표를 선택할 수 있다.
  • 모델 결과를 업무 문장으로 해석할 수 있다.

6. 종합 실습 문제

1. 문제 출제

고객 데이터에서 다음 두 문제를 각각 정의하고 모델링 전략을 작성하세요.

  • 문제 A: 다음 달 예상 매출 예측
  • 문제 B: 다음 달 구매 여부 예측

2. 문제 해결에 필요한 개념 요약

문제유형목표값평가 지표
다음 달 예상 매출회귀next_month_salesMSE
구매 여부분류purchased정확도

3. 수도코드

데이터를 읽는다. 입력 컬럼을 정한다. 문제 A의 목표값을 정한다. 회귀 모델을 학습하고 MSE를 계산한다. 문제 B의 목표값을 정한다. 분류 모델을 학습하고 정확도를 계산한다. 두 결과를 비교하고 해석 문장을 작성한다.

4. 실행 예시

문제 A 유형: 회귀 문제 A 평가: MSE = 1200000.0 문제 B 유형: 분류 문제 B 평가: 정확도 = 0.8 해석: 매출 금액 예측은 오차 크기를 확인하고, 구매 여부 예측은 맞힌 비율과 틀린 유형을 함께 확인해야 합니다.
