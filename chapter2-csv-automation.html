Chapter 2-1

CSV 자동화 처리

여러 CSV 파일을 한 번에 읽고, 하나로 합친 뒤 합계와 평균을 계산하는 자동화 흐름을 학습합니다.

메인Chapter 2다음: 엑셀 파일 처리 1

1. 왜 CSV 자동화가 필요한가?

실무에서는 월별 매출 파일, 지점별 주문 파일, 날짜별 로그 파일처럼 같은 형식의 CSV가 여러 개로 나뉘어 있는 경우가 많습니다. 파일을 하나씩 열어 복사해서 붙이는 방식은 실수가 많고 시간이 오래 걸립니다.

파일 목록 찾기glob으로 여러 CSV 파일을 찾습니다.
반복 읽기반복문으로 파일을 하나씩 읽습니다.
데이터 합치기pd.concat()으로 하나의 표로 합칩니다.
요약 계산합계, 평균, 건수 같은 지표를 계산합니다.

2. 실습 폴더 구조

여러 CSV 파일을 자동으로 처리하려면 파일이 일정한 폴더에 정리되어 있어야 합니다.

project/ ├── data/ │ ├── sales_2026_01.csv │ ├── sales_2026_02.csv │ └── sales_2026_03.csv └── notebooks/
권장: 원본 데이터는 data/raw, 처리 결과는 data/processed처럼 폴더를 나누면 관리가 쉽습니다.

3. 여러 CSV 파일 찾기

glob.glob()을 사용하면 특정 패턴에 맞는 파일 목록을 한 번에 가져올 수 있습니다.

import glob file_list = glob.glob("data/sales_*.csv") print(file_list)

*는 여러 글자를 대신하는 와일드카드입니다. sales_*.csv는 sales_로 시작하고 .csv로 끝나는 모든 파일을 의미합니다.

4. 반복문으로 파일 읽기

파일 목록을 만든 뒤 반복문으로 하나씩 읽고 리스트에 저장합니다.

import pandas as pd frames = [] for file_path in file_list: df = pd.read_csv(file_path) frames.append(df)

리스트에 DataFrame을 모아두면 나중에 한 번에 합칠 수 있습니다.

5. CSV 파일 합치기

pd.concat()은 여러 DataFrame을 세로 방향으로 이어붙일 때 사용합니다.

sales_all = pd.concat(frames, ignore_index=True) print(sales_all.head()) print(sales_all.shape)
핵심: ignore_index=True를 사용하면 기존 인덱스를 무시하고 0부터 새 인덱스를 붙입니다.

6. 합계와 평균 계산

파일을 합친 뒤에는 컬럼 단위로 합계와 평균을 계산할 수 있습니다.

total_sales = sales_all["sales"].sum() average_sales = sales_all["sales"].mean() count_rows = len(sales_all) print("총매출:", total_sales) print("평균 매출:", average_sales) print("데이터 건수:", count_rows)

7. 그룹별 집계

카테고리나 지점별로 매출을 보고 싶다면 groupby()를 사용합니다.

summary = sales_all.groupby("branch")["sales"].sum() print(summary)

그룹별 집계는 Chapter 2 전체에서 계속 사용되는 핵심 문법입니다.

8. 실습 체크리스트

  • glob으로 여러 CSV 파일을 찾을 수 있다.
  • 반복문으로 여러 CSV 파일을 읽을 수 있다.
  • pd.concat()으로 DataFrame을 합칠 수 있다.
  • 합계, 평균, 건수를 계산할 수 있다.
  • groupby()로 그룹별 매출을 계산할 수 있다.

9. 종합 실습 문제

1. 문제 출제

월별 매출 CSV 3개를 하나로 합치고, 지점별 총매출을 계산한 뒤 결과를 새 CSV 파일로 저장하세요.

  • data/sales_2026_01.csv, data/sales_2026_02.csv, data/sales_2026_03.csv 파일을 읽습니다.
  • 세 파일을 하나의 DataFrame으로 합칩니다.
  • branch별 sales 합계를 계산합니다.
  • 결과를 branch_sales_summary.csv로 저장합니다.

2. 문제 해결에 필요한 개념 요약

개념활용 방법
glob파일 목록을 자동으로 찾습니다.
for 문파일을 하나씩 반복해서 읽습니다.
concat여러 DataFrame을 합칩니다.
groupby지점별 총매출을 계산합니다.
to_csv결과를 파일로 저장합니다.

3. 수도코드

CSV 파일 목록을 찾는다. 빈 리스트를 만든다. 파일 목록을 반복하면서 각 CSV를 읽는다. 읽은 DataFrame을 리스트에 추가한다. 리스트 안의 DataFrame을 하나로 합친다. branch 기준으로 sales 합계를 계산한다. 결과를 화면에 출력한다. 결과를 CSV 파일로 저장한다.
