Chapter 2이전다음

1. 엑셀 파일 구조 이해

엑셀 파일은 하나의 통합 문서 안에 여러 워크시트가 들어 있는 구조입니다. CSV는 보통 하나의 표만 담지만, 엑셀은 여러 시트와 서식, 수식까지 포함할 수 있습니다.

Workbook엑셀 파일 전체입니다.
Worksheet파일 안의 개별 시트입니다.
Cell행과 열이 만나는 하나의 칸입니다.
Header표의 컬럼 이름입니다.

2. 엑셀 파일 읽기

pd.read_excel()을 사용하면 엑셀 파일을 DataFrame으로 읽을 수 있습니다.

import pandas as pd df = pd.read_excel("sales.xlsx") print(df.head())
설치 필요: 엑셀 파일을 읽을 때 openpyxl이 필요할 수 있습니다. 설치 명령은 pip install openpyxl입니다.

3. 특정 시트 읽기

엑셀 파일 안에 여러 시트가 있을 때는 sheet_name 옵션을 사용합니다.

df = pd.read_excel("sales.xlsx", sheet_name="1월매출") print(df.head())

  • 엑셀 통합 문서와 워크시트 차이를 설명할 수 있다.

  • sheet_name으로 특정 시트를 읽을 수 있다.
  • CSV와 엑셀 파일의 차이를 설명할 수 있다.

6. 종합 실습 문제

1. 문제 출제

monthly_sales.xlsx 파일의 1월매출 시트를 읽고, 데이터 구조를 확인하세요.

  • 앞부분 5행을 출력합니다.
  • 행과 열 개수를 출력합니다.
  • 컬럼명을 출력합니다.
  • 매출 컬럼의 총합을 계산합니다.
