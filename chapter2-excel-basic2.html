1. 이번 단원에서 배우는 것

실무 엑셀 파일은 월별, 지점별, 부서별 시트로 나뉘어 있는 경우가 많습니다. 이번 단원에서는 여러 시트를 자동으로 읽고 하나의 데이터로 정리하는 흐름을 연습합니다.

시트명 컬럼 추가출처 시트를 데이터에 남깁니다.
여러 파일 합치기파일 단위 반복 처리로 확장합니다.

2. 시트 목록 확인

pd.ExcelFile()을 사용하면 파일 안의 시트 목록을 확인할 수 있습니다.

import pandas as pd excel_file = pd.ExcelFile("sales.xlsx") print(excel_file.sheet_names)

3. 여러 시트 읽기

시트 이름 목록을 반복하면서 각 시트를 읽고 리스트에 저장합니다. 시트명도 하나의 컬럼으로 추가하면 나중에 출처를 알 수 있습니다.

frames = [] for sheet in excel_file.sheet_names: df = pd.read_excel("sales.xlsx", sheet_name=sheet) df["sheet_name"] = sheet frames.append(df) sales_all = pd.concat(frames, ignore_index=True) print(sales_all.head())

4. 시트별 필터링

읽은 데이터를 하나로 합친 뒤 특정 시트나 특정 조건에 맞는 데이터만 선택할 수 있습니다.

january = sales_all[sales_all["sheet_name"] == "1월"] high_sales = sales_all[sales_all["sales"] >= 100000] print(january.head()) print(high_sales.head())

5. 여러 엑셀 파일 합치기

glob과 read_excel()을 함께 사용하면 폴더 안의 여러 엑셀 파일을 자동으로 합칠 수 있습니다.

import glob files = glob.glob("data/*.xlsx") frames = [] for file in files: df = pd.read_excel(file) df["source_file"] = file frames.append(df) result = pd.concat(frames, ignore_index=True)
주의: 여러 파일을 합칠 때는 컬럼명이 동일한지 먼저 확인해야 합니다.

6. 실습 체크리스트

  • 엑셀 파일의 시트 목록을 확인할 수 있다.
  • 여러 시트를 반복해서 읽을 수 있다.
  • 시트명을 컬럼으로 추가할 수 있다.
  • 조건에 맞는 데이터만 필터링할 수 있다.
  • 여러 엑셀 파일을 하나로 합칠 수 있다.
