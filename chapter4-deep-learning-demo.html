Chapter 4이전다음

1. MNIST란?

MNIST는 손글씨 숫자 이미지 데이터셋입니다. 0부터 9까지의 숫자 이미지를 보고 어떤 숫자인지 맞히는 분류 문제에 사용됩니다.

입력손글씨 숫자 이미지
출력0~9 중 하나의 숫자
문제 유형다중 분류
학습 목표이미지 패턴 인식

2. 이미지 데이터의 구조

컴퓨터는 이미지를 그림으로 보는 것이 아니라 숫자 배열로 봅니다. 흑백 이미지는 각 픽셀의 밝기 값으로 표현할 수 있습니다.

이미지 한 장 = 픽셀 숫자들의 모음 28 × 28 이미지 = 784개의 숫자

3. ReLU 활성화 함수

ReLU는 음수는 0으로 만들고, 양수는 그대로 통과시키는 단순한 활성화 함수입니다. 딥러닝에서 은닉층에 자주 사용됩니다.

입력값이 0보다 작으면 0 입력값이 0보다 크면 입력값 그대로 출력
직관: ReLU는 약한 신호를 끄고, 강한 신호는 살려서 다음 층으로 전달하는 역할을 합니다.

4. CNN, 풀링, 드롭아웃

개념역할쉬운 설명
CNN이미지 특징 추출선, 모서리, 패턴을 찾습니다.
풀링정보 압축중요 특징만 남기고 크기를 줄입니다.
드롭아웃오버피팅 완화학습 중 일부 뉴런을 잠시 꺼서 과도한 암기를 줄입니다.

5. 케라스 데모 코드 흐름

실제 코드는 복잡해 보일 수 있지만, 흐름은 Chapter 3의 머신러닝과 비슷합니다.

데이터 불러오기 이미지 숫자값 정규화 모델 만들기 모델 학습 테스트 데이터 평가 예측 결과 확인
핵심: 딥러닝도 결국 “데이터 준비 → 모델 학습 → 예측 → 평가” 흐름으로 진행됩니다.

7. 종합 실습 문제

1. 문제 출제

MNIST 숫자 인식 문제를 딥러닝 프로젝트 관점에서 설명하는 설계서를 작성하세요.

  • 입력 데이터가 무엇인지 작성합니다.
  • 출력 결과가 무엇인지 작성합니다.
  • 문제 유형을 작성합니다.
  • CNN, 풀링, 드롭아웃이 각각 어떤 역할을 하는지 설명합니다.
  • 모델 학습 흐름을 단계별로 정리합니다.

2. 문제 해결에 필요한 개념 요약

개념활용 방법
MNIST손글씨 숫자 이미지 데이터
다중 분류0~9 중 하나를 예측

3. 수도코드

MNIST 데이터를 불러온다. 이미지 데이터를 정규화한다. CNN 모델 구조를 만든다. 모델을 컴파일한다. 훈련 데이터로 학습한다. 테스트 데이터로 평가한다. 예측 결과를 확인한다. 결과를 문장으로 해석한다.

4. 실행 예시

입력: 28×28 손글씨 숫자 이미지 출력: 0~9 중 예측 숫자 문제 유형: 다중 분류 평가 예시: 테스트 정확도 0.98 해석: 모델이 테스트 이미지의 98%를 올바르게 분류했습니다.
