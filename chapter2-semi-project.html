1. 프로젝트 목표

여러 월의 매출 데이터를 하나로 통합하고, 결측치와 자료형을 정리한 뒤 카테고리·지점별 매출을 분석하여 그래프와 해석 문장이 포함된 미니 리포트를 작성합니다.

2. 문제 출제

월별 매출 CSV 파일 3개를 하나로 합쳐 매출 현황 EDA 리포트를 작성하세요.

  • 여러 CSV 파일을 자동으로 읽습니다.
  • 파일을 하나의 DataFrame으로 합칩니다.
  • 결측치와 자료형을 점검하고 처리합니다.
  • 전체 매출, 평균 매출, 최고 매출 카테고리를 계산합니다.
  • 카테고리별·지점별 총매출을 계산합니다.
  • 카테고리별 총매출 그래프와 지점별 총매출 그래프를 작성합니다.
  • 분석 결과를 5문장 이상의 리포트로 정리합니다.

5. 수도코드

필요한 라이브러리를 불러온다. 월별 CSV 파일 목록을 찾는다. 각 CSV 파일을 반복해서 읽는다. 읽은 DataFrame을 리스트에 저장한다. 여러 DataFrame을 하나로 합친다. 데이터 구조와 결측치를 확인한다. 필요한 결측치와 자료형을 처리한다. 전체 매출과 평균 매출을 계산한다. 카테고리별 총매출을 계산한다. 지점별 총매출을 계산한다. 두 개의 막대그래프를 작성한다. 정제 데이터를 CSV로 저장한다. 분석 리포트를 텍스트 파일로 저장한다.
