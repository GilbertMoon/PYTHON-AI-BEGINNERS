Chapter 3-1

머신러닝 개념

머신러닝, 지도학습, 비지도학습, 데이터에서 규칙을 찾는 과정을 초보자 관점에서 이해합니다.

메인Chapter 3다음: 머신러닝 수학 직관

1. 머신러닝이란?

머신러닝은 사람이 규칙을 일일이 작성하는 대신, 컴퓨터가 데이터에서 패턴을 찾아 예측이나 분류를 수행하게 하는 방법입니다.

입력 데이터모델이 참고하는 정보입니다.
정답 데이터모델이 맞혀야 하는 값입니다.
학습입력과 정답의 관계를 찾는 과정입니다.
예측새로운 입력에 대해 결과를 추정합니다.

2. 머신러닝 문제 유형

구분의미예시
지도학습정답이 있는 데이터로 학습가격 예측, 합격/불합격 분류
비지도학습정답 없이 데이터 구조 탐색고객 군집화
강화학습행동 결과의 보상으로 학습게임 AI, 로봇 제어

3. 지도학습의 두 가지 핵심 문제

이 과정에서는 지도학습 중에서도 가장 기본적인 회귀와 분류를 중심으로 다룹니다.

구분예측 결과예시
회귀연속적인 숫자매출, 가격, 온도 예측
분류정해진 클래스합격/불합격, 구매/미구매
핵심 구분: 예측 결과가 숫자 크기라면 회귀, 종류나 라벨이라면 분류로 생각하면 됩니다.

4. 머신러닝 전체 흐름

  1. 문제를 정의합니다.
  2. 데이터를 수집합니다.
  3. 데이터를 전처리합니다.
  4. 입력값 X와 정답 y를 나눕니다.
  5. 모델을 학습합니다.
  6. 새 데이터로 예측합니다.
  7. 성능을 평가하고 해석합니다.
데이터 준비 → 전처리 → X/y 분리 → 모델 학습 → 예측 → 평가 → 해석

5. 실습 체크리스트

  • 머신러닝의 의미를 설명할 수 있다.
  • 지도학습, 비지도학습, 강화학습을 구분할 수 있다.
  • 회귀와 분류의 차이를 설명할 수 있다.
  • 머신러닝 프로젝트의 전체 흐름을 말할 수 있다.

6. 종합 실습 문제

1. 문제 출제

아래 업무 사례를 보고 회귀 문제인지, 분류 문제인지, 비지도학습 문제인지 구분해 보세요.

  • 고객의 다음 달 구매 금액 예측
  • 고객이 이탈할지 여부 예측
  • 비슷한 소비 패턴을 가진 고객 그룹 찾기
  • 아파트 면적과 위치를 이용한 가격 예측

2. 문제 해결에 필요한 개념 요약

개념판단 기준
회귀결과가 연속적인 숫자
분류결과가 정해진 종류
비지도학습정답 없이 구조나 그룹을 찾음

3. 수도코드

각 사례의 예측 결과를 확인한다. 결과가 숫자인지 라벨인지 판단한다. 정답 데이터가 있는지 확인한다. 회귀, 분류, 비지도학습 중 하나로 분류한다. 각 판단 이유를 한 문장으로 작성한다.

4. 실행 예시

고객의 다음 달 구매 금액 예측 → 회귀 고객 이탈 여부 예측 → 분류 소비 패턴 기반 고객 그룹 찾기 → 비지도학습 아파트 가격 예측 → 회귀
